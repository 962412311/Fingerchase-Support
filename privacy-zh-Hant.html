指尖逐音隱私權政策

生效日期：2026 年 7 月 18 日

1. 適用範圍與提供者

本政策適用於指尖逐音 App 及其官方支援網頁。本政策中的「我們」指 App Store 所示的 App 提供者 Parsifal Chen。Apple、GitHub、從 App 開啟的外部網站，以及你在系統分享面板中選擇的目標，會依據各自的條款與隱私權政策獨立處理資訊。

2. 隱私權概要

指尖逐音以裝置端運作為主要原則。目前版本中，我們沒有營運用於接收或儲存你的練習歷史、個人樂譜、匯入圖片、PDF、音訊檔案、麥克風音訊或辨識結果的伺服器。但當你使用 Apple 訂閱服務、開啟外部連結、瀏覽支援網頁、提交 GitHub 支援要求或透過其他 App 分享文字時，仍可能發生網路通訊。

3. 儲存在裝置上的資訊

App 可能在裝置上儲存練習與瀏覽歷史、收藏、設定、首次引導與每日目標選擇、體驗及權益計數、永久解鎖曲目、個人樂譜中繼資料與檔案、轉換結果、鋼琴校準資料和辨識最佳化參數。這些資訊只用於提供你主動使用的功能、復原裝置端狀態、個人化練習，以及執行免費體驗或訂閱權益，不會透過我們營運的 App 伺服器傳送給我們。

4. 檔案與使用者提供的內容

結構化樂譜、圖片、PDF 和音訊只會在你透過系統檔案選擇器明確選取後存取。App 可能將所選檔案複製到自身的私有儲存空間、建立暫時處理檔案，並儲存衍生樂譜或轉換結果。個人內容不會上傳給我們。App 不為個人樂譜、音符資料、來源檔案或錄音提供公開連結、匯出、列印或分享功能。你只應匯入自己有權使用的內容，並避免匯入你認為高度敏感的資訊。

5. 麥克風處理

只有當你主動開始真實鋼琴跟彈、校準或辨識最佳化時，App 才會要求麥克風權限。功能運作期間，音訊幀在裝置上即時分析；原始麥克風音訊不會上傳給我們，也不會由 App 儲存為錄音。衍生的校準或最佳化參數可能保留在裝置上，直到你刪除。由於啟用中的麥克風可能收錄附近聲音，請只在合適的環境使用，並在私人談話前停止該功能。

6. 訂閱、購買與診斷

購買、續訂、退款、付款憑證、反詐欺處理與 Apple 帳號資訊由 Apple 處理。App 只透過 StoreKit 接收產品、交易驗證、訂閱狀態與到期時間等有限資訊，以提供 Pro 權益；我們不會取得你的完整信用卡資料。依據你的裝置和 Apple 隱私權設定，Apple 也可能向我們提供有限的當機報告或彙整診斷，以便修正缺陷。Apple 獨立控制其資訊收集、安全與保留。

7. 我們不收集的資訊

目前版本不要求建立指尖逐音帳號，也不索取你的姓名、電子郵件、聯絡人、精確位置、健康資訊、生物辨識資訊，或對照片圖庫與檔案庫的無限制存取。我們不整合第三方廣告或分析 SDK、不使用廣告識別碼、不跨其他公司的 App 或網站追蹤你、不建立廣告側寫、不出售個人資訊，也不使用個人資料作出會產生法律或類似重大影響的決定。

8. 第三方、外部連結與分享

Apple 提供 App Store 與訂閱服務。本支援網站由 GitHub Pages 託管；GitHub 明確說明會基於安全目的記錄訪客 IP 位址。GitHub Issue 是公開內容，且可能需要 GitHub 帳號，因此絕不能在其中提交密碼、付款資料、個人樂譜、錄音或其他敏感資訊。開啟樂譜來源或其他外部網站時，瀏覽器要求及 IP 位址等資訊會傳送給該網站。當你主動使用系統分享面板分享練習成果文字或邀請文字時，你選擇的目標會依自身隱私權規則接收這些文字。指尖逐音不會自動向這些第三方傳送個人樂譜內容。

9. 處理目的與法律依據

在適用法律要求說明法律依據的地區，裝置端處理用於提供你主動要求的功能和履行 App 服務，並以你的選擇及必要權限為依據；訂閱處理用於提供你要求的購買；有限診斷可能基於維護安全與可靠性的正當利益；法律要求時也可能處理相關資訊。你可以隨時撤回麥克風權限，或停止使用選用的檔案、連結、支援及分享功能。撤回不影響撤回前已完成的合法處理。

10. 保留、刪除與備份

你可以在 App 內刪除個人樂譜及其相關練習資料、校準資料和辨識最佳化資料。其他裝置端歷史、收藏、設定與權益狀態會保留到 App 資料被刪除或 App 被解除安裝。解除安裝會從該裝置移除 App 目前使用的裝置端容器，但既有裝置備份或雲端備份可能依據你的 Apple 備份設定與 Apple 的保留規則繼續保存副本，並可能在復原備份時再次出現。訂閱記錄由 Apple 保留，刪除 App 不會取消訂閱。GitHub 和外部網站會依據自身政策保留日誌或你提交的內容；公開支援 Issue 在透過 GitHub 編輯或刪除前可能持續可見。

11. 你的選擇與權利

你可以在系統設定中撤回麥克風權限、在 App 內刪除支援刪除的裝置端內容、解除安裝 App 以刪除其餘目前裝置端資料、透過 Apple 管理或刪除相關備份、在 Apple 帳號中管理訂閱，並選擇不開啟外部連結或不使用分享。由於我們沒有儲存裝置端內容的 App 伺服器副本，通常無法代你存取、匯出、更正或刪除這些內容。依所在地法律，你可能對實際由某個控制者持有的個人資訊享有知情、存取、更正、刪除、限制、反對、可攜或撤回同意等權利。涉及 Apple、GitHub 或其他目標的資訊，應直接向相應提供者提出要求。

12. 安全措施與剩餘風險

我們透過 App 範圍內的私有儲存、系統權限控制、明確的檔案選擇與裝置端處理降低風險，但任何裝置、軟體、備份或傳輸方式都無法保證絕對安全。剩餘風險包括：未鎖定、共用、遺失、遭入侵或被備份的裝置被他人存取；內容暴露給你主動選擇的分享目標；資訊公開在 GitHub Issue；作業系統或服務提供者發生安全事件；惡意或格式異常的匯入檔案；以及裝置端資料損毀或誤刪。請使用裝置密碼、保持系統更新、檢查備份與分享設定、為重要檔案保留獨立且合法的副本，且不要透過公開支援管道提交敏感內容。

13. 兒童隱私

指尖逐音不要求帳號、不提供定向廣告，也不會明知而透過我們營運的 App 伺服器收集兒童個人資訊。監護人應依當地法律監督未成年人的裝置使用、檔案匯入、麥克風使用、外部連結、分享與購買。Apple 提供適用的購買及家庭控制。

14. 跨境處理

我們不營運 App 內容伺服器，也不會將你的個人樂譜、練習歷史或麥克風音訊跨境傳輸。Apple、GitHub、外部網站與分享目標可能依據各自隱私權聲明與法律保障，在其他國家或地區處理資訊。

15. 政策變更與聯絡方式

在引入重大資料處理變更之前或同時，我們會更新本政策及生效日期。如有一般隱私權問題，請使用我們的公開支援頁面。GitHub Issue 是公開內容，請不要填寫任何個人或敏感資訊。裝置端 App 資料應使用上述控制方式自行管理；由 Apple 或 GitHub 持有的資訊必須透過相應提供者處理。
