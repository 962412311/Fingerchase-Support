指尖逐音隐私政策

生效日期：2026 年 7 月 18 日

1. 适用范围与提供者

本政策适用于指尖逐音 App 及其官方支持网页。本政策中的“我们”指 App Store 所示的 App 提供者 Parsifal Chen。Apple、GitHub、从 App 打开的外部网站以及你在系统分享面板中选择的目标，会依据各自的条款和隐私政策独立处理信息。

2. 隐私概要

指尖逐音以本机运行为主要原则。当前版本中，我们没有运营用于接收或保存你的练习历史、个人曲谱、导入图片、PDF、音频文件、麦克风音频或识别结果的服务器。但当你使用 Apple 订阅服务、打开外部链接、访问支持网页、提交 GitHub 支持请求或通过其他 App 分享文字时，仍可能发生网络通信。

3. 保存在设备上的信息

App 可能在设备上保存练习和浏览历史、收藏、设置、首次引导和每日目标选择、体验及权益计数、永久解锁曲目、个人曲谱元数据与文件、转换结果、钢琴校准数据和识别优化参数。这些信息只用于提供你主动使用的功能、恢复本机状态、个性化练习以及执行免费体验或订阅权益，不会通过我们运营的 App 服务器发送给我们。

4. 文件与用户提供的内容

结构化曲谱、图片、PDF 和音频只会在你通过系统文件选择器明确选取后访问。App 可能把所选文件复制到自身的私有存储空间、创建临时处理文件并保存派生曲谱或转换结果。个人内容不会上传给我们。App 不为个人曲谱、音符数据、源文件或录音提供公开链接、导出、打印或分享功能。你只应导入自己有权使用的内容，并避免导入你认为高度敏感的信息。

5. 麦克风处理

只有当你主动开始真实钢琴跟弹、校准或识别优化时，App 才会请求麦克风权限。功能运行期间，音频帧在设备上实时分析；原始麦克风音频不会上传给我们，也不会被 App 保存为录音。派生的校准或优化参数可能保留在设备上，直到你删除。由于启用中的麦克风可能捕捉附近声音，请仅在合适的环境中使用，并在私人谈话前停止该功能。

6. 订阅、购买与诊断

购买、续订、退款、付款凭据、反欺诈处理和 Apple 账户信息由 Apple 处理。App 仅通过 StoreKit 接收产品、交易验证、订阅状态和到期时间等有限信息，以提供 Pro 权益；我们不会取得你的完整银行卡资料。根据你的设备和 Apple 隐私设置，Apple 还可能向我们提供有限的崩溃报告或汇总诊断，以便修复缺陷。Apple 独立控制其信息收集、安全和保留。

7. 我们不收集的信息

当前版本不要求创建指尖逐音账户，也不索取你的姓名、电子邮件、通讯录、精确位置、健康信息、生物识别信息，或对照片库和文件库的无限制访问。我们不集成第三方广告或分析 SDK，不使用广告标识符，不跨其他公司的 App 或网站追踪你，不建立广告画像，不出售个人信息，也不使用个人数据作出会产生法律或类似重大影响的决定。

8. 第三方、外部链接与分享

Apple 提供 App Store 和订阅服务。本支持站点由 GitHub Pages 托管；GitHub 明确说明会出于安全目的记录访问者 IP 地址。GitHub Issue 是公开内容，且可能要求 GitHub 账户，因此绝不能在其中提交密码、付款资料、个人曲谱、录音或其他敏感信息。打开曲谱来源或其他外部网站时，浏览器请求及 IP 地址等信息会发送给该网站。当你主动使用系统分享面板分享练习成果文字或邀请文字时，你选择的目标会依据自身隐私规则接收这些文字。指尖逐音不会自动向这些第三方发送个人曲谱内容。

9. 处理目的与法律依据

在适用法律要求说明法律依据的地区，本机处理用于提供你主动请求的功能和履行 App 服务，并以你的选择及必要权限为依据；订阅处理用于提供你请求的购买；有限诊断可能基于维护安全与可靠性的正当利益；法律要求时也可能处理相关信息。你可以随时撤回麦克风权限，或停止使用可选的文件、链接、支持及分享功能。撤回不影响撤回前已经完成的合法处理。

10. 保留、删除与备份

你可以在 App 内删除个人曲谱及其关联练习数据、校准数据和识别优化数据。其他本机历史、收藏、设置和权益状态会保留到 App 数据被删除或 App 被卸载。卸载会从该设备移除 App 当前使用的本机容器，但已有设备备份或云备份可能依据你的 Apple 备份设置和 Apple 的保留规则继续保存副本，并可能在恢复备份时重新出现。订阅记录由 Apple 保留，删除 App 不会取消订阅。GitHub 和外部网站会依据自身政策保留日志或你提交的内容；公开支持 Issue 在通过 GitHub 编辑或删除前可能持续可见。

11. 你的选择与权利

你可以在系统设置中撤回麦克风权限，在 App 内删除支持删除的本机内容，卸载 App 以删除其余当前本机数据，通过 Apple 管理或删除相关备份，在 Apple 账户中管理订阅，并选择不打开外部链接或不使用分享。由于我们没有保存本机内容的 App 服务器副本，通常无法替你访问、导出、更正或删除这些内容。根据所在地法律，你可能对实际由某个控制者持有的个人信息享有知情、访问、更正、删除、限制、反对、可携带或撤回同意等权利。涉及 Apple、GitHub 或其他目标的信息，应直接向相应提供者提出请求。

12. 安全措施与剩余风险

我们通过 App 范围内的私有存储、系统权限控制、明确的文件选择和本机处理降低风险，但任何设备、软件、备份或传输方式都无法保证绝对安全。剩余风险包括：未锁定、共用、遗失、遭入侵或被备份的设备被他人访问；内容暴露给你主动选择的分享目标；信息公开在 GitHub Issue；操作系统或服务提供者发生安全事件；恶意或格式异常的导入文件；以及本机数据损坏或误删。请使用设备密码、保持系统更新、检查备份和分享设置、为重要文件保留独立且合法的副本，并且不要通过公开支持渠道提交敏感内容。

13. 儿童隐私

指尖逐音不要求账户、不提供定向广告，也不会明知而通过我们运营的 App 服务器收集儿童个人信息。监护人应根据当地法律监督未成年人的设备使用、文件导入、麦克风使用、外部链接、分享和购买。Apple 提供适用的购买及家庭控制。

14. 跨境处理

我们不运营 App 内容服务器，也不会把你的个人曲谱、练习历史或麦克风音频跨境传输。Apple、GitHub、外部网站和分享目标可能依据各自隐私声明与法律保障，在其他国家或地区处理信息。

15. 政策变更与联系

在引入重大数据处理变化之前或同时，我们会更新本政策及生效日期。如有一般隐私问题，请使用我们的公开支持页面。GitHub Issue 是公开内容，请不要填写任何个人或敏感信息。本机 App 数据应使用上述控制方式自行管理；由 Apple 或 GitHub 持有的信息必须通过相应提供者处理。
